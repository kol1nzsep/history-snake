# word-snake

This is a fun project started by me in May 23, 2026, for my history class presentation.
Although if you find it interesting and you want to run it on your own personal computer, here is the installation guide for this project:

1. Clone the repo.
2. Install python and pip on your computer.
3. Open terminal and type: pip install pygame.
4. Click on the dist folder and on the game.exe file.
5. Success! Enjoy.
